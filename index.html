route-tokenizer

Path tokenizer for routers

Getting Started

Installing

npm install @thoughtsunificator/route-tokenizer

Usage

import { Tokenizer } from "@thoughtsunificator/route-tokenizer"


const tokens = Tokenizer.tokenize("/account/{id}")
